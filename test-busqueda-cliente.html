Test: Búsqueda de Cliente por Cédula

Funcionalidad a Probar

Este test verifica la nueva funcionalidad de búsqueda de cliente por cédula/RUC:

  1. Buscar cliente existente por cédula
  2. Mostrar información del cliente encontrado
  3. Crear nuevo cliente si no existe
  4. Integración con el módulo de ventas
Pasos para Probar
  1. Abre el dashboard del vendedor
  2. Ve al módulo de ventas
  3. En "Detalles de Venta", busca un cliente por cédula
  4. Si existe, verifica que se muestre su información
  5. Si no existe, prueba crear uno nuevo
Resultados Esperados
  • ✅ Campo de búsqueda por cédula/RUC
  • ✅ Búsqueda con Enter o botón
  • ✅ Información del cliente encontrado
  • ✅ Opción para crear cliente nuevo
  • ✅ Integración con materiales de reparación
